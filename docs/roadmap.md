# SzymOS Kernel Roadmap

This roadmap describes planned features and milestones for **SzymOS Kernel**.

---

## Current Version: 0.2.0
- VGA terminal driver
- GDT initialization
- Multiboot-compliant ISO
- QEMU test setup

---

## Short-Term Goals (v0.3 – v0.5)
- Keyboard input driver
- Simple command prompt / shell
- Basic file I/O (in-memory filesystem)
- Interrupt handling (IDT)

---

## Medium-Term Goals (v0.6 – v1.0)
- Memory management
- Hardware detection modules
- Modular kernel library support
- Improved error handling and logs

---

## Long-Term Goals (v1.x+)
- Preemptive multitasking
- PCI / USB support
- Simple GUI / windowing system
- Network stack (TCP/IP)
- Expandable kernel modules

---

## Contribution Opportunities
- Writing drivers
- Improving kernel core functions
- Adding documentation
- Testing across QEMU and real hardware
- Proposing new modules or features
